O Bootcamp TQI Fullstack Developer é um programa gratuito para formação em desenvolvimento fullstack e recrutamento de desenvolvedores juniores pela TQI.

O curso ele é divido por módulos, ao qual cada módulo aborda um tema de programação contendo atividades para serem realizadas durante o ensino.

Módulo I - Conteúdo de abertura (6 atividades)

Módulo II - HTML/CSS3 (2 atividades)

Módulo III - Javascript (12 atividades)

Módulo IV - ReactJS (6 atividades)

Módulo V - SQL (2 atividades)

Módulo VI - Java (15 atividades)

Módulo VII - Spring Framework (5 atividades)

Módulo VIII - QA (3 atividades)

Assim que você acaba os cursos você obtém o certificado de conhecimento podendo assim ampliar os horizontes em relação a programação.
